# battleship
клієнт-серверна гра "Морський бій" на Python та C++

## Технології:
- **Frontend**: Python (бібліотека Tkinter)
- **Backend**: C++ (структурне програмування)
- **Зв'язок**: IPC через стандартні потоки (stdin/stdout)
- **Синхронізація**: Threading Lock

## Як запустити:
1. Завантажте папку з проєктом.
2. Переконайтеся, що у вас встановлено Python.
3. Запустіть файл `start_game.bat`.
